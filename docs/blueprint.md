# **App Name**: Estudaqui PWA

## Core Features:

- Overview Dashboard: Dashboard with visual cards for subjects, progress, and status, highlighting the next subject and estimated time. Displays time studied (today, week, month, total) in graphs, daily average, streak, and subject distribution. Also shows total subjects, completed subjects, cycles completed, and average time per subject.
- Revision Sequence: Review system with a fixed sequence, displaying the current subject, progress, and controls to mark as complete, skip, or pause, logging all review history. Includes a timer and a queue of upcoming subjects, and calculates optimal review times by observing usage as a tool.
- Pomodoro Timer: Pomodoro timer with customizable focus, short break, long break, and cycle times, plus notification sounds. Features a visible timer, session controls (play, pause, reset, skip), and session counters. A persistent widget shows time remaining and current state across all tabs.
- Study Cycle Management: Management of subjects with auto-numbering (starting from 0) and drag-and-drop reordering, allowing users to add, remove, and edit subjects/topics, plus search, filters, and export/import functionalities.
- Persistent Pomodoro Widget: Persistent widget always visible showing remaining time, the state (Focus/Pause) and a play/pause button. Clicking the widget redirects to the Pomodoro tab.

## Style Guidelines:

- Primary color: Sky blue (#4DB6AC) to create a calm and focused environment.
- Background color: Very light cyan (#E0F7FA). Background and primary are the same hue, and the background is light to complement the saturated primary color.
- Accent color: Mellow green (#A5D6A7), as a neighboring hue to give enough contrast for important calls to action.
- Body and headline font: 'Inter', a sans-serif with a clean, neutral design, well-suited for both headlines and body text.
- Simple, line-based icons to represent different subjects and actions, ensuring clarity and ease of understanding.
- Mobile-first responsive design, ensuring a seamless experience across devices. Use clear and concise layouts for each section.
- Subtle transitions and animations to enhance user experience without being distracting.
